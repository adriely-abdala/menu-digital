# 📱 Menu Digital

Aplicação web desenvolvida para exibição de cardápio digital, facilitando a visualização de produtos de forma prática e organizada.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📌 Funcionalidades
- Exibição de itens do cardápio
- Interface simples e intuitiva
- Layout responsivo

## 📅 Ano
2026

## 🎯 Objetivo
Projeto desenvolvido com o objetivo de praticar conceitos de desenvolvimento web e criar uma solução funcional de cardápio digital.

## ▶️ Como executar
1. Baixe ou clone o repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador
